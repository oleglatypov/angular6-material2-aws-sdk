# speed-dial

## Usage

### default
```
<stbui-speed-dial>
    <button mat-mini-fab>
        <mat-icon>search</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>edit</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>home</mat-icon>
    </button>
</stbui-speed-dial>
```

### setting open properties 
```
<stbui-speed-dial open="true">
    <button mat-mini-fab>
        <mat-icon>search</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>edit</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>home</mat-icon>
    </button>
</stbui-speed-dial>
```
open value
* false
* true

### setting position properties
```
<stbui-speed-dial position="down">
    <button mat-mini-fab>
        <mat-icon>search</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>edit</mat-icon>
    </button>
    <button mat-mini-fab>
        <mat-icon>home</mat-icon>
    </button>
</stbui-speed-dial>
```
position value
* up
* down
* left
* right
